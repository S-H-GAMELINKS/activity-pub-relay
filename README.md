# ActivityPub Relay

Mastodon and Misskey(or other Misskey fork) supported ActivityPub Relay.
Also support Pleroma and Akkoma.

## Requirements

- Ruby 3.4.1
- Rails 8.0.0
- SQLite3
- Docker

You do not need Redis, Nginx, nor PostgreSQL to deploy this Relay.
