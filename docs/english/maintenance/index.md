## Maintenance
### Watch logs

```console
kamal app logs -f
```

### Open Rails console

```console
kamal console

# Or

kamal app exec bin/rails c -i
```

### Open Server console

```console
kamal app exec bash -i
```

### Check configuration

```console
kamal config
# Output configurations like a IP address, SSH port info... 
```

### Deploy

```console
kamal deploy
```
